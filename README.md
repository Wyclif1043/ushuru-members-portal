    # Ushuru Investment Co-operative - Members Portal

A comprehensive web application for members of Ushuru Investment Co-operative Society to manage their investments, track contributions, and communicate with the society in real-time.

## Features

- **Secure Member Dashboard** - Overview of contributions and investments
- **Profile Management** - KYC and document upload
- **Contributions Tracking** - View balances and history
- **Investment Management** - Apply and track investments
- **Refunds & Financial Operations** - Request refunds and download receipts
- **Communication Portal** - Secure messaging and support tickets
